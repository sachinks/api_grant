## Columns

All output files share the same 9-column schema:

| Column | Description | Example |
|--------|-------------|---------|
| FIRST | Investigator first name | Jane |
| LAST | Investigator last name | Smith |
| INSTITUTION | Awarding institution | HARVARD UNIVERSITY |
| TITLE | Grant title | Study of Neural Plasticity |
| FUNDING_AMT | Award amount (number) | 500000 |
| CURRENCY | Currency code | USD / EUR / GBP |
| AWARD_DATE | Date award was issued | 2025-01-10 |
| SOURCE | Data source name | NIH RePORTER |
| LINK | Direct URL to grant record | https://reporter.nih.gov/... |
